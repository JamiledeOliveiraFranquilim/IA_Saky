### Como Começar?

Para fazer este projeto florescer no seu ambiente, siga estes passos:

#### 1\. Pré-Requisitos Necessários

Antes de tudo, certifique-se de que você tem:

  * **VSCODE** (versão [X.X] ou superior)
  * **Extensão de Python e biblioteca de Python**
  * **Chaves API de IA**

#### 2\. Instalação e Preparação

1.  **Clone o Repositório:** Copie o link do código deste repositório
   
2.  **Entre na Pasta:**
    Crie una pasta com o arquivo nomeado como: "AI_Saky". Logo, abra o terminal e use o comando "Git Clone"
    e copie o link do código

#### 3\. Para Rodar o Projeto
 Apenas clique no play, no canto superior direito da tela.

### Contribuições (Você é Bem-Vindo\!)

Sua ajuda é como um raio de sol para o nosso projeto\! Se você encontrou um pequeno erro, tem uma ideia brilhante para uma nova funcionalidade, ou quer apenas adicionar um toque de cor, por favor:

1.  Faça um *fork* (uma "cópia") do projeto.
2.  Crie uma *branch* com um nome descritivo (`git checkout -b feature/sua-ideia`).
3.  Faça suas mudanças e **documente-as com clareza**.
4.  Crie um *Pull Request* (PR) para a *branch* principal (`main`).
